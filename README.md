# Employee Burnout Prediction (ML Project)

This project predicts employee burnout risk (Burn Rate) using regression techniques and HR analytics data such as workload, mental fatigue score, designation, company type, and WFH setup.

The goal is to identify employees at risk of burnout early so organizations can take preventive actions to improve well-being and productivity.

---

## 📌 Dataset

Kaggle Dataset: *Are Your Employees Burning Out?*

Features include:

* Gender
* Company Type
* WFH Setup
* Designation Level
* Resource Allocation (workload)
* Mental Fatigue Score
* Burn Rate (target)

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🧠 ML Workflow

1. Data Cleaning & Handling Missing Values
2. Feature Engineering & Encoding
3. Train-Test Split & Scaling
4. Model Training using Linear Regression
5. Model Evaluation using RMSE, MAE, and R²

---

## 📊 Evaluation Metrics

* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* Mean Absolute Error (MAE)
* R² Score
